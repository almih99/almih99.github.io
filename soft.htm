Построение графа условной диагностики

Программа предназначена для построения графа последовательной условной программы технической диагностики по таблице функций неисправностей с помощью функции предпочтения.

Была написана на 4 курсе института вместо экзамена по диагностике САУ. Главным требованием было обеспечить возможность запуска на компьютере с 286 процессором. Тогда я был настолько наглым, что самостоятельно для данной программы написал что-то вроде простой оконной системы...

Сейчас, конечно, её можно запустить только при помощи dosbox или dosemu.
